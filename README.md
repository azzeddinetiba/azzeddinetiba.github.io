# azzeddinetiba.github.io

Personal academic website of **Azzeddine Tiba** — Computational R&D Engineer at [Michelin](https://www.michelin.com/), PhD in Applied Mathematics / Scientific Machine Learning from [CNAM](https://www.cnam.fr/) (Paris).

**Live site:** [https://azzeddinetiba.github.io](https://azzeddinetiba.github.io)

## About

This website showcases my research work, publications, talks, teaching activities, software contributions, and CV. My research lies at the intersection of scientific computing, machine learning, and computational mechanics, with a focus on data-driven reduced order models and ML-enhanced numerical methods for fluid-structure interaction.

## Running Locally

This site is built with [Jekyll](https://jekyllrb.com/) using the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template.

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve -l -H localhost
```

The site will be available at `http://localhost:4000`. Alternatively, use Docker:

```bash
docker compose up
```
